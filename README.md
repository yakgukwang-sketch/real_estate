# Seoul Real Estate & Commercial District Simulation

서울시 상권 및 부동산 시장을 시뮬레이션하는 프로젝트입니다.

## 프로젝트 개요

서울시의 상권 데이터와 부동산 시장 데이터를 결합하여 분석하고, 지역별 상권 활성도와 부동산 가격 간의 상관관계를 시뮬레이션합니다.

## 주요 기능

- 서울시 상권 데이터 수집 및 분석 (업종별, 지역별)
- 부동산 실거래가 데이터 수집 및 분석
- 상권 활성도와 부동산 가격 상관관계 분석
- 지역별 상권 변화 시뮬레이션
- 시각화 대시보드
